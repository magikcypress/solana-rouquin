# 05 — Creer et animer la communaute

## La regle numero 1

> Un meme coin sans communaute, c'est juste du code. La communaute EST le produit.

DOGE vaut des milliards non pas a cause de sa tech mais parce que des millions de personnes trouvent ca drole. $ROUQ doit suivre la meme logique.

## Les plateformes essentielles

### Twitter / X (obligatoire)

C'est LA plateforme crypto. 90% du buzz meme coin se passe sur Twitter.

**Setup :**
1. Creer `@RouquinSol`
2. Bio : "Malin comme un Rouquin $ROUQ | Premier meme coin FR sur Solana | Community-driven"
3. Banner : version large du logo avec le ticker
4. Lien : vers le site web
5. Pinned tweet : presentation du projet avec le CA (contract address)

**Strategie de contenu :**

| Frequence | Type | Exemple |
|-----------|------|---------|
| Quotidien | Meme | Image/video avec le renard |
| Quotidien | GM post | "GM la meute" avec emoji renard |
| 3x/semaine | Update | "Phase 1 complete, on passe a la meute" |
| 2x/semaine | Engagement | Sondage, question, concours |
| 1x/semaine | Thread | Explication du projet, roadmap update |

**Hashtags :** #ROUQ #RouquinSol #SolanaMeme #memecoin #Solana

**Astuce :** Les reponses aux gros comptes crypto (Ansem, Blknoiz06, etc.) avec des memes $ROUQ bien places = visibilite gratuite.

### Telegram (obligatoire)

Le "QG" de la communaute. C'est la que les holders discutent.

**Setup :**
1. Creer le groupe `t.me/RouquinSOL`
2. Ajouter un bot anti-spam (Combot ou Rose)
3. Message fixe : regles + liens + CA
4. Ajouter les admins/mods

**Regles du groupe :**
- Pas de FUD (Fear, Uncertainty, Doubt)
- Pas de promotion d'autres tokens
- Pas de scam/phishing
- Memes encourages

**Bots utiles :**
- **BuyBot** : notification quand quelqu'un achete du ROUQ (cree du FOMO)
- **PriceBot** : /price pour voir le cours en direct
- **WalletBot** : /balance pour checker son solde

### Discord (optionnel, phase 2)

Plus structure que Telegram. Utile quand la communaute depasse 500+ membres.

Channels suggeres :
```
#annonces
#general
#memes
#prix-et-trading
#suggestions
#support
```

## Strategie de lancement : les 7 premiers jours

### Jour 0 (pre-lancement)
- Creer Twitter + Telegram
- Poster un teaser : logo + "Something foxy is coming to Solana"
- Inviter 10-20 personnes de confiance

### Jour 1 (lancement)
- Annoncer le token sur Twitter (thread)
- Ouvrir le Telegram au public
- Partager le site web
- Poster sur Reddit (r/solana, r/CryptoMoonShots, r/SatoshiStreetBets)

### Jour 2-3
- Premier concours de memes (#RouquinMemes)
- Engager avec les comptes crypto FR (comusol, Le Journal du Coin, etc.)
- Poster le CA partout

### Jour 4-5
- Annoncer l'airdrop OG (premiers 100 membres Telegram)
- Faire des raids Twitter (la communaute like/RT en masse)

### Jour 6-7
- Pool de liquidite sur Raydium
- Annonce officielle du trading live
- Celebrer les premiers trades

## KOLs (Key Opinion Leaders)

Les influenceurs crypto peuvent booster enormement la visibilite.

### KOLs crypto FR
| Nom | Plateforme | Followers | Note |
|-----|-----------|-----------|------|
| Hasheur | YouTube/Twitter | 600k+ | Selectif, contenu edu |
| Le Journal du Coin | Site/Twitter | 100k+ | Media crypto FR |
| CryptoMatrix | YouTube | 200k+ | Trading-oriented |
| Comusol | Twitter | Communaute FR Solana | |

### Approche
- **Ne jamais payer un KOL avec des tokens unlock** — ils vont dump
- Proposer un partenariat : contenu + allocation lock 6 mois
- Les micro-influenceurs (1k-10k followers) sont souvent plus efficaces et moins chers
- Offrir des tokens pour des reviews honnetes (pas des shills)
